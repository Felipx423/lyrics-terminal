# Prompt Base para Codex

Leia antes de alterar o projeto:

- docs/Visao-Geral.md
- docs/Arquitetura.md
- docs/Decisoes.md
- docs/Bugs.md
- docs/Roadmap.md

Regras:
- Não alterar decisões arquiteturais sem atualizar docs/Decisoes.md.
- Se corrigir bug, atualizar docs/Bugs.md.
- Se implementar feature, atualizar docs/Roadmap.md.
- Fazer alterações em branch separada.
- Commitar código e documentação juntos quando fizer sentido.
- Não fazer merge na main sem aprovação.
